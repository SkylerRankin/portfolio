Performance Graphing

Provides a visualization to see if algorithms follow 'big o notation' as they would in theory. The tests are as follows: for constant performance, a truth check O(1); for linear performance, a linear search O(n); for quadratic performance, a duplicate value check on two arrays O(n^2); for exponential performance decay, Fibbonaci number calculation O(2^n); and finally for logarithmic performance, a binary Search on sorted list O(log(n)).
github
